# 作業開始前に実行すること

1. **Docker Desktop の起動**
2. **ターミナルの quiz-app 配下で以下のコマンドを実行**

```
sail up -d
```

3. **以下のコマンドを実行**

```
sail npm run dev
```

4. **localhost を開く**

# 作業終了時に実行すること

1. **sail npm run dev を実行したタブで ctrl+C を入力**
2. **quiz-app 配下で以下のコマンド実行**

```
sail down
```

4. **Docker Desktop の終了**
